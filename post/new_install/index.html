Setting up your new Mac for academic work

Wed, Jun 7, 2017 new installation

This is mainly for my own use, but I figured it might be useful to put it on-line…

When changing computer, it’s always a pain in the ass to set up your new machine like the old one. Over time, I tried to gather the different commands for installing the various programs I use to ease the setup of my next computer (or the update of my current one). You’ll find the following entries on my github page, as well as my .bash_profile and my .Rprofile files.

  • Xcode Command Line Tools
  • Homebrew
  • GIT
  • Gfortran and GCC
  • XQuartz
  • GNU commands
  • xmgrace and gnuplot
  • ImageMagick
  • LaTeX
  • PYTHON
  • R
  • LAPACK
  • LAMMPS
  • Sublime Text
  • List of my Applications
  • To have no time to wait when pointing on the hidden Dock
  • Write screenshots as pdf
  • Academic website with hugo
